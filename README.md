**TP de SLAM PHP et SQL**

Ce TP a été fait dans le cadre de la formation SLAM en BTS SIO. Il consiste à mettre en place un projet PHP suivant le modèle MVC, suite aux compétences acquises en PHP.

Dans ce projet, une base de données MySQL/MariaDB est manipulée afin de gérer les produits et la liste fournisseurs d'une entreprise fictive, au travers d'un site intéractif.

Le front-end du site a été conçu avec du bootstrap pour moderniser l'interface.
